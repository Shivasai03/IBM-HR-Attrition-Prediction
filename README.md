# 📊 IBM HR Analytics — Employee Attrition Prediction

A machine learning project that predicts employee attrition using the IBM HR Analytics dataset.

## 🎯 Objective
Identify employees likely to leave the company using classification models, and extract actionable business insights.

## 📁 Dataset
- 1,470 employees | 35 features | 0 missing values
- Attrition rate: 16.1%

## 🛠️ Tech Stack
Python | Pandas | Scikit-learn | Matplotlib | Seaborn | Jupyter Notebook

## 🤖 Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting

## 📌 Key Findings
- OverTime is the strongest driver of attrition
- Sales department has the highest attrition rate
- Low income employees are significantly more likely to leave
- Single employees leave at a higher rate
